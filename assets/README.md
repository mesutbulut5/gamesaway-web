# Assets Klasörü

Bu klasör, web sitesinin medya dosyalarını içerir.

## Gerekli Dosyalar

### 1. gamesaway-demo.mp4
- **Açıklama**: Ana sayfa hero bölümünde gösterilecek tanıtım videosu
- **Kaynak**: https://www.4usgames.com/kopyas%C4%B1-4us-world
- **Boyut**: Maksimum 5MB (web için optimize edilmiş)
- **Format**: MP4 (H.264)
- **Çözünürlük**: 720p veya 1080p
- **Süre**: 10-30 saniye

### Video İndirme
`VIDEO-INDIRME.md` dosyasına bakın.

### 2. logo.png (İsteğe Bağlı)
- GamesAway logosu
- Boyut: 200x200px
- Format: PNG (şeffaf arka plan)

### 3. favicon.ico (İsteğe Bağlı)
- Site ikonu
- Boyut: 32x32px veya 64x64px
- Format: ICO veya PNG

## Kullanım

Videoyu indirdikten sonra:
1. `gamesaway-demo.mp4` olarak bu klasöre kaydedin
2. Tarayıcıda `index.html` dosyasını açın
3. Video otomatik olarak oynatılacak

## Not

Video bulunamazsa, mevcut placeholder (telefon ikonu) gösterilecektir.
